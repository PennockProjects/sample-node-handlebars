## Sample Application Using a Node and Handlebars Templates

Requirements

Install [Express.js](http://github.com/visionmedia/express) Module

```
npm install express
```

Install [hbs](https://github.com/donpark/hbs) module

```
npm install hbs
```


## Running Server ##

```
$ node app.js
```

## Testing Server ##

Visit http://localhost:1337/
